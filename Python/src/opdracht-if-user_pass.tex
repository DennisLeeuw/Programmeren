\begin{itemize}
\item Schrijf een script dat vraagt om een gebruikersnaam.
\item Als er iets is ingevoerd vraag dan om een wachtwoord.
\item Als ook het wachtwoord is gegeven print beide dan op het scherm en vraag de gebruiker dan om bevestiging.
\item Als de gebruiker \textquote{ja} heeft gegegeven eindigt het script met de tekst \textquote{Juiste invoer}
\item Als de gebruiker \textquote{nee} heeft gegeven eindigt het script met de tekst \textquote{Incorrect invoer}
\end{itemize}
