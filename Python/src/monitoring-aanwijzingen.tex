We gebruiken \texttt{ping} om te testen of de server er is, we testen op het \textquote{open} zijn van de poorten 80 en 443 om te controleren of de webserver functioneert en we ondervragen de webserver met de \texttt{Host}-header om te zien of de domeinen operationeel zijn. Je zal dus moeten beginnen met uit te zoeken hoe dit gedaan kan worden binnen Python. Tevens zal je je moeten verdiepen in hoe HTTP werkt.

Tot slot: het is waarschijnlijk handig om zelf een web-server op te zetten waar je de tool tegen kan testen.
