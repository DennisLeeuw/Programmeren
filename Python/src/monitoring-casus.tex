Binnen de organisatie hebben we een webserver waarop meerdere domeinen gehost zijn. De eigenaren van de domeinen mogen zelf hun sites bijhouden en er wijzigingen in aanbrengen. Het zelf wijzigingen aanbrengen heeft soms tot gevolg dat een domein onbereikbaar wordt. Meestal is dit een foutje (bug) in de PHP code. Ook de beheerders van de server maken weleens fouten in het beheren van de server zodat dat soms heel de webserver eruit ligt en een heel enkele keer moet er onderhoud worden gedaan en dan is de server aan het rebooten en dus helemaal niet bereikbaar.
