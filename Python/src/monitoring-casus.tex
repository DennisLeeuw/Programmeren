We hebben op onze webserver meerdere domeinen gehost. De server is een Raspberry PI met daarop het Raspbian (Linux) besturingssysteem. Als webserver gebruiken we Apache en we hebben er 3 domeinen op draaien voor onze familieleden:
\begin{itemize}
\item karel.nova.local
\item merel.nova.local
\item peter.nova.local
\end{itemize}

De gebruikers mogen zelf hun domein bijhouden en er wijzigingen op aanbrengen. Het zelf wijzigingen aanbrengen heeft soms tot gevolg dat een domein onbereikbaar wordt. Ook wij maken weleens fouten in het beheren van de server zodat dat soms heel Apache eruit ligt en een heel enkele keer moet er onderhoud worden gedaan en dan is de Raspberry PI aan het rebooten.
