We hebben tot nu toe elke keer een regel weggeschreven in rapport.txt of notities.txt die bestond uit de verzamelde data gescheiden door een komma. Een bestand met \textquote{Comma Separated Values} heet een CSV-bestand. Voor CSV bestanden gelden meer regels dan alleen dat de waarden gescheiden moeten zijn door een komma. Om te voldoen aan al die regels is er een Python module met de naam \texttt{csv} die ervoor zorgt dat de weggeschreven data echt een CSV bestand vormt.

Voor nadere uitleg over deze module zie het \texttt{python\_csv.pdf} bestand.

\begin{enumerate}
\item Installeer de CSV-module
\item Maak een kopie van je notities applicatie
\item Herschrijf de kopie van de applicatie aan zodat de output een echt CSV bestand is en noem dit output bestand notities.csv.
\item Maak een kopie van je rapportage applicatie
\item Herschrijf de kopie van de applicatie zodat de output een echt CSV bestand is en noem dit output bestand rapport.csv.
\end{enumerate}
