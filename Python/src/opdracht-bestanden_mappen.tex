Laat het programma een map met bestanden inlezen en een overzicht maken in report.txt.
\begin{enumerate}
\item Vraag aan de gebruiker om een map
\item Test of de map bestaat
\item Haal informatie op van alle bestanden in de map
\item Schrijf in report.txt: naam van de map,bestandsnaam,grootte,datum
\item Bonus: Maak een notitie in notes.txt met de naam van de map die geanalyseerd is, met natuurlijk de datum en tijd.
\end{enumerate}
