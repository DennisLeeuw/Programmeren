De te maken tool moet aan de volgende eisen voldoen:
\begin{itemize}
\item er moet gebruik gemaakt worden van variabelen
\item er moet gebruik gemaakt zijn van het if-statement
\item er moet gebruik gemaakt zijn van minimaal 1 loop
\item er moet minimaal 1 eigen functie gemaakt zijn
\item er moet minimaal 1 module gebruikt zijn
\item problemen die kunnen ontstaan tijdens het gebruik van de tool moeten worden afgevangen
\end{itemize}
