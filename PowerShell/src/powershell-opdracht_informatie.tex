Maak een script dat:
\begin{enumerate}
\item Een lijst met computernamen accepteerd als argument
\item Een argument heeft om een bepaald stuk informatie op te vragen
\item Haal van de computer informatie op
\item Schrijf de data weg als een JSON bestand
\end{enumerate}

Het script moet de volgende informatie (minimaal) kunnen opvragen:
\begin{itemize}
\item Versienummer van Windows
\item IP-adres van de netwerkkaarten
\item Beschikbare lokale gebruikers op het systeem
\end{itemize}
